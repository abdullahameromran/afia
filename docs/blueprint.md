# **App Name**: Afia Balsemak

## Core Features:

- User Input Form: Collects the user's name, age, and question about women's health through a simple form.
- Intelligent Q&A: Uses AI to analyze the user's question against a database of frequently asked questions and keywords, providing a relevant answer. A tool is used to filter unwanted text in case the response will violate company policy.
- Personalized Response Display: Displays the AI-generated response clearly, addressing the user by name to personalize the interaction.
- Extensible Knowledge Base: Utilize a database that covers a range of common women's health inquiries, with the ability to easily expand the database.

## Style Guidelines:

- Primary color: Dusty rose (#D81B60) to evoke a sense of caring and femininity, mirroring the health focus for women.
- Background color: Very light pink (#FCE4EC) to create a soft and inviting backdrop.
- Accent color: Light pastel purple (#E0BBE4) used for interactive elements, such as buttons, to provide visual contrast.
- Headline font: 'Belleza' (sans-serif) for headlines; body font: 'Alegreya' (serif). Using Belleza for a personal feel and Alegreya to ensure that the potentially lengthy AI responses will be easy to read.
- Use delicate, hand-drawn style icons related to health and wellness to complement the design.
- Maintain a clean and simple layout with a focus on readability and user-friendly design, optimized for mobile devices.
- Subtle animations when displaying the AI response to signal processing.
